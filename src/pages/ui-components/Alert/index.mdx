# Alert

Pode ser utilizado para mostrar mensagens de atualizações para os usuários, sejam atualizações de segurança ou
de interface. O objetivo é avisar e não guiar o caminho para o usuário.

### Importação

```js
import Alert from '@eduzz/houston-ui/Alert';
```

### Exemplo

<Playground>
  <Alert title='Atenção!!!' actionProps={{ text: 'Ação customizada' }}>
    Alert pode ser utilizado para mostrar mensagens de atualizações para os usuários, sejam atualizações de segurança ou
    de interface. O objetivo é avisar e não guiar o caminho para o usuário.
  </Alert>
</Playground>

### Gerenciável

Controle a exibição pela prop `visible`.

<Playground>
  {() => {
    const [visible, setVisible] = React.useState(true);
    const toggle = () => setVisible(v => !v);
    return (
      <>
        <Box xs={{ margin: '5px 0' }}>
          <Alert visible={visible}>
            Alert pode ser utilizado para mostrar mensagens de atualizações para os usuários, sejam atualizações de
            segurança ou de interface. O objetivo é avisar e não guiar o caminho para o usuário.
          </Alert>
        </Box>
        <Button onClick={toggle}>Mostrar/esconder</Button>
      </>
    );
  }}
</Playground>

### Ação

É possível adicionar apenas uma ação (CTA). A prop `actionProps` disponibiliza todas as props do componente Button.

<Playground>
  <Alert actionProps={{ text: 'Ação customizada', onClick: () => alert('Houston ❤️') }}>
    Alert pode ser utilizado para mostrar mensagens de atualizações para os usuários, sejam atualizações de segurança ou
    de interface. O objetivo é avisar e não guiar o caminho para o usuário.
  </Alert>
</Playground>

### Alert props

| prop        | tipo              | obrigatório | padrão        | descrição                                                 |
| ----------- | ----------------- | ----------- | ------------- | --------------------------------------------------------- |
| type        | `AlertTypes`      | `false`     | `informative` | -                                                         |
| title       | `React.ReactNode` | `false`     | -             | -                                                         |
| closeIcon   | `boolean`         | `false`     | `true`        | Se `false`, o ícone (x) será ocultado.                    |
| visible     | `boolean`         | `false`     | -             | Controle de exibição.                                     |
| actionProps | `AlertAction`     | `false`     | -             | Se houver valor na prop `text`, será renderizado o botão. |
| onClose     | `function`        | `false`     | -             | -                                                         |
