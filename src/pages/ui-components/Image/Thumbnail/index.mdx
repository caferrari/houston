# Thumbnail

As thumbnails são versões reduzidas de imagens, usadas para tornar mais fácil o processo de procura, localização e reconhecimento.

### Importação

```js
import Thumbnail from '@eduzz/houston-ui/Image/Thumbnail';
```

## Propriedades

Possui todas propriedades do componente [Image](/houston/ui-components/Image) e:

| prop | tipo                     | obrigatório | padrão | descrição |
| ---- | ------------------------ | ----------- | ------ | --------- |
| size | `'sm', 'md', 'lg', 'xl'` | `false`     | `md`   | -         |
