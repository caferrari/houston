# Tag

O componente Tag é usado para itens que precisam ser rotulados, categorizados ou organizados usando palavras-chave. Também pode ser usado como um botão que controla estados.

### Importação

```js
import Tag from '@eduzz/houston-ui/Tag';
```

### Exemplo

<Playground>
  <Tag onClick={() => console.log('click on the Tag')} onClose={() => console.log('click on X')}>
    <Tag.Left onClick={() => console.log('click on TagLeft')}>
      <Add size='sm' />
    </Tag.Left>
    <Tag.Text>Tag</Tag.Text>
  </Tag>
</Playground>

### Tag props

| prop     | tipo       | obrigatório | padrão  | descrição                                         |
| -------- | ---------- | ----------- | ------- | ------------------------------------------------- |
| disabled | `boolean`  | `false`     | `false` | -                                                 |
| isActive | `boolean`  | `false`     | `false` | -                                                 |
| onClose  | `Function` | `false`     | `false` | Se passado o componente vai renderizar um ícone X |
| onClick  | `Function` | `false`     | `false` | -                                                 |

### Tag.Left props

| prop     | tipo              | obrigatório | padrão  | descrição |
| -------- | ----------------- | ----------- | ------- | --------- |
| children | `React.ReactNode` | `true`      | -       | -         |
| onClick  | `Function`        | `false`     | `false` | -         |

### Tag.Text props

| prop     | tipo              | obrigatório | padrão | descrição |
| -------- | ----------------- | ----------- | ------ | --------- |
| children | `React.ReactNode` | `true`      | -      | -         |

### Tag.Right props

| prop     | tipo              | obrigatório | padrão  | descrição |
| -------- | ----------------- | ----------- | ------- | --------- |
| children | `React.ReactNode` | `true`      | -       | -         |
| onClick  | `Function`        | `false`     | `false` | -         |
