# TagCategory

A Tag Category serve para categorizar um determinado conteúdo, deixando claro que ele pertence a um entre vários tipos.

### Importação

```js
import TagCategory from '@eduzz/houston-ui/Tag/Category';
```

### Exemplo

<Playground>
  <Row>
    <Column>
      <TagCategory>
        <FaceHappyOutline size='sm' />

      </TagCategory>
    </Column>
    <Column>
      <TagCategory>TagCategory</TagCategory>
    </Column>
  </Row>
</Playground>
