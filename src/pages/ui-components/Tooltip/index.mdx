# Tooltip

Usado para identificar elementos e descrever funcionalidades.

### Importação

```js
import Tooltip from '@eduzz/houston-ui/Tooltip';
```

### Exemplo

<Playground>
  <Row>
    <Column>
      <Tooltip title='Lorem ipsum dolor'>
        <Button>John Doe</Button>
      </Tooltip>
    </Column>
    <Column>
      <Tooltip title='Lorem ipsum dolor' disabled>
        <Button>Tooltip desabilitado</Button>
      </Tooltip>
    </Column>
  </Row>
</Playground>

### Direção

<Playground>
  <Row>
    <Column>
      <Tooltip title='Lorem ipsum dolor'>
        <Button>Top</Button>
      </Tooltip>
    </Column>
    <Column>
      <Tooltip title='Lorem ipsum dolor' placement='bottom'>
        <Button>Bottom</Button>
      </Tooltip>
    </Column>
    <Column>
      <Tooltip title='Lorem ipsum dolor' placement='left'>
        <Button>Left</Button>
      </Tooltip>
    </Column>
    <Column>
      <Tooltip title='Lorem ipsum dolor' placement='right'>
        <Button>Right</Button>
      </Tooltip>
    </Column>
  </Row>
</Playground>

| prop      | tipo                | obrigatório | padrão  | descrição |
| --------- | ------------------- | ----------- | ------- | --------- |
| title     | `string`            | `true`      | -       | -         |
| disabled  | `boolean`           | `false`     | `false` | -         |
| placement | `ITooltipPlacement` | `false`     | `top`   | -         |
| onOpen    | `function`          | `false`     | -       | -         |
| onClose   | `function`          | `false`     | -       | -         |
