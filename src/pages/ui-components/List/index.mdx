# Lista

Listas são continuas, apresentam verticalmente texto ou imagens.

### Importação

```js
import List from '@eduzz/houston-ui/List';
```

### List.Item props

| prop     | tipo              | obrigatório | padrão | descrição |
| -------- | ----------------- | ----------- | ------ | --------- |
| children | `React.ReactNode` | `true`      | -      | -         |

### List.Left props

| prop     | tipo              | obrigatório | padrão | descrição |
| -------- | ----------------- | ----------- | ------ | --------- |
| icon     | `React.ReactNode` | `false`     | -      | -         |
| children | `React.ReactNode` | `true`      | -      | -         |

### List.Text props

| prop     | tipo                        | obrigatório | padrão | descrição |
| -------- | --------------------------- | ----------- | ------ | --------- |
| title    | `string or React.ReactNode` | `true`      | -      | -         |
| subtitle | `string or React.ReactNode` | `fase`      | -      | -         |

### List.Right props

| prop     | tipo              | obrigatório | padrão | descrição |
| -------- | ----------------- | ----------- | ------ | --------- |
| icon     | `React.ReactNode` | false       | -      | -         |
| onClick  | `function`        | false       | -      | -         |
| children | `React.ReactNode` | `true`      | -      | -         |
