# Progress

O componente é usado para exibir o status de andamento.

### Importação

```ts
import Progress from '@eduzz/houston-ui/Progress';
```

### Exemplo

<Playground>
  {() => {
    const [value, setValue] = useState(50);
    const add = () => setValue(prevState => prevState + 10);
    const remove = () => setValue(prevState => prevState - 10);
    return (
      <>
        <Box xs={{ padding: '0.5rem 0' }}>
          <Progress value={value} showInfo={false} />
        </Box>
        <Box xs={{ padding: '0.5rem 0' }}>
          <Progress value={value} />
        </Box>
        <Box xs={{ padding: '0.5rem 0' }}>
          <Progress value={value} infoPosition='right' />
        </Box>
        <div style={{ gap: 16, display: 'flex', marginTop: '1rem' }}>
          <Button onClick={() => add()}>+ 10</Button>
          <Button onClick={() => remove()}>- 10</Button>
        </div>
      </>
    );
  }}
</Playground>

### Exemplo com labels

É possível inserir label em todos os cantos do componente.

> Atenção, quando uma das labels está preenchida, o valor atual (%) não é exibido.

<Playground>
  <Box xs={{ padding: '20px 0' }}>
    <Progress
      labelTopStart='Info Top'
      labelTopEnd='Info Top'
      labelBottomStart='Info Bottom'
      labelBottomEnd='Info Bottom'
      value={50}
    />
  </Box>
</Playground>

| prop             | tipo                   | obrigatório | padrão       | descrição                                                       |
| ---------------- | ---------------------- | ----------- | ------------ | --------------------------------------------------------------- |
| value            | `number`               | `true`      | -            | -                                                               |
| max              | `number`               | `false`     | `100`        | -                                                               |
| min              | `number`               | `false`     | `0`          | -                                                               |
| showInfo         | `boolean`              | `false`     | `true`       | Exibição da quantidade preenchida.                              |
| infoPosition     | `ProgressInfoPosition` | `false`     | `left`       | -                                                               |
| format           | `ProgressFormat`       | `false`     | `percentage` | Se `number`, o símbolo de porcentagem (%) não será renderizado. |
| labelTopStart    | `React.ReactNode`      | `false`     | -            | -                                                               |
| labelTopEnd      | `React.ReactNode`      | `false`     | -            | -                                                               |
| labelBottomStart | `React.ReactNode`      | `false`     | -            | -                                                               |
| labelTopEnd      | `React.ReactNode`      | `false`     | -            | -                                                               |
