# Button Icon

Botões permitem que os usuários tomem ações e decisões com um simples toque.

### Importação

```js
import ButtonIcon from '@eduzz/houston-ui/ButtonIcon';
```

### Exemplos

<Playground>
  <ButtonIcon>
    <InsertEmoticonIcon />
  </ButtonIcon>
  <ButtonIcon disabled>
    <InsertEmoticonIcon />
  </ButtonIcon>
</Playground>

### Tamanho

<Playground>
  <ButtonIcon size='medium'>
    <InsertEmoticonIcon />
  </ButtonIcon>
  <ButtonIcon size='large'>
    <InsertEmoticonIcon />
  </ButtonIcon>
</Playground>

| prop     | tipo              | obrigatório | padrão  | descrição |
| -------- | ----------------- | ----------- | ------- | --------- |
| disabled | `boolean`         | `false`     | `false` | -         |
| size     | `medium`, `large` | `false`     | `large` | -         |
