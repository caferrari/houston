# Showcase

O componente Showcase é utilizado para exibir informações de forma interativa ao usuário.

### Importação

```ts
import Showcase from '@eduzz/houston-ui/Showcase';
```

### Showcase

| prop                 | tipo                       | obrigatório | padrão  | descrição                                              |
| -------------------- | -------------------------- | ----------- | ------- | ------------------------------------------------------ |
| open                 | `boolean`                  | `false`     | `true`  | Controla o estado aberto/fechado do componente         |
| initialStep          | `number`                   | `false`     | 1       | Indica em qual passo o componente será renderizado     |
| onNext               | `Function`                 | `false`     | -       | -                                                      |
| onPrevious           | `Function`                 | `false`     | -       | -                                                      |
| onFinish             | `Function`                 | `false`     | -       | -                                                      |
| onClose              | `Function`                 | `false`     | -       | -                                                      |
| stepCounter          | `boolean`                  | `false`     | `true`  | Se `true`, exibe o contador de passos totais e o atual |
| disableBackdropClick | `boolean`                  | `false`     | `true`  | -                                                      |
| size                 | `small`, `medium`, `large` | `false`     | `large` | Define a largura do componente                         |

### Showcase.Image

| prop | tipo     | obrigatório | padrão | descrição |
| ---- | -------- | ----------- | ------ | --------- |
| src  | `string` | `true`      | -      | -         |
| alt  | `string` | `true`      | -      | -         |

### Showcase.StepButtons

| prop            | tipo      | obrigatório | padrão  | descrição                                          |
| --------------- | --------- | ----------- | ------- | -------------------------------------------------- |
| hideCloseButton | `boolean` | `false`     | `false` | Se `true`, não exibe o botão fechar no passo atual |

### Showcase.GenericButtons

| prop            | tipo      | obrigatório | padrão  | descrição                                         |
| --------------- | --------- | ----------- | ------- | ------------------------------------------------- |
| hideCloseButton | `boolean` | `false`     | `false` | Se true, não exibe o botão fechar em nenhum passo |

### Showcase.PreviousButton
