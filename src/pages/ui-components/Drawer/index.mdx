# Drawer

Um painel que desliza da borda da tela.

### Importação

```js
import Drawer from '@eduzz/houston-ui/Drawer';
```

### Exemplo

É possível usar o Drawer sem o cabeçalho e rodapé.

<Playground>
  {() => {
    const [visible, setVisible] = React.useState(false);
    const onOpen = () => setVisible(true);
    const onClose = () => setVisible(false);
    return (
      <>
        <Drawer visible={visible} onClose={onClose}>
          <Drawer.Header>Exemplo Drawer</Drawer.Header>
          <Drawer.Content>
            O Drawer é um menu usado em aplicações web e mobile que, de início, fica fora da visão do usuário e só
            aparece através de uma instrução.
            <br />
            <br />
            <img src='/houston/img/welcome.png' width='100%' />
          </Drawer.Content>
          <Drawer.Footer>
            <Button onClick={onClose}>Fechar</Button>
          </Drawer.Footer>
        </Drawer>
        <Button onClick={onOpen}>Abrir</Button>
      </>
    );
  }}
</Playground>

### Drawer props

| prop             | tipo              | obrigatório | padrão  | descrição                                                               |
| ---------------- | ----------------- | ----------- | ------- | ----------------------------------------------------------------------- |
| visible          | `boolean`         | `true`      | `-`     | Controle de exibição.                                                   |
| placement        | `DrawerPlacement` | `false`     | `right` | Posicionamento do elemento.                                             |
| size             | `ModalSize`       | `false`     | `sm`    | -                                                                       |
| closeIcon        | `boolean`         | `false`     | `true`  | Se `false`, o ícone (x) será ocultado.                                  |
| onClose          | `function`        | `true`      | -       | -                                                                       |
| disableEscapeKey | `boolean`         | `false`     | `-`     | Se `true`, ao pressionar a tecla ESC não é acionado a função `onClose`. |

### Drawer Header props

| prop              | tipo      | obrigatório | padrão  | descrição                                                  |
| ----------------- | --------- | ----------- | ------- | ---------------------------------------------------------- |
| disableTypography | `boolean` | `false`     | `false` | Se `true`, o componente `Typography` não será renderizado. |
