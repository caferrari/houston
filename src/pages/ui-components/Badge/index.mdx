# Badge

Pequeno valor numérico ou descritor de status para elementos da interface do usuário.

### Importação

```js
import Badge from '@eduzz/houston-ui/Badge';
```

### Exemplo

<Playground>
  <IconButton>
    <Badge>
      <Icon />
    </Badge>
  </IconButton>
  <IconButton>
    <Badge count={9}>
      <Icon />
    </Badge>
  </IconButton>
  <IconButton>
    <Badge count={101}>
      <Icon />
    </Badge>
  </IconButton>
</Playground>

### Cores disponíveis

Valores disponíveis: `informative`, `positive`, `negative` e `warning`. Padrão `informative`.

### Distâncias

Em alguns casos, você pode diminuir o espaço entre a Badge e o elemento. Valores disponíveis: `xs`, `sm` e `md`. Padrão `md`.

<Playground>
  <div style={{ display: 'inline-flex', gap: 16 }}>
    <Badge count={101} distance='xs'>
      <Button>Meu botão</Button>
    </Badge>
    <Badge count={101} distance='sm'>
      <Button>Meu botão</Button>
    </Badge>
    <Badge count={101} distance='md'>
      <Button>Meu botão</Button>
    </Badge>
  </div>
</Playground>

### Alert props

| prop     | tipo            | obrigatório | padrão        | descrição                                  |
| -------- | --------------- | ----------- | ------------- | ------------------------------------------ |
| count    | `number`        | `false`     | -             | -                                          |
| color    | `BadgeColors`   | `false`     | `informative` | -                                          |
| dot      | `boolean`       | `false`     | `false`       | Se `true`, o número interno será ocultado. |
| distance | `BadgeDistance` | `false`     | `md`          | Distância entre o elemento com a Badge.    |
