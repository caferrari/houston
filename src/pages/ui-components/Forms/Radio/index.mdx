# Radio

O componente de Radio é utilizado quando é preciso deixar todas as opções visíveis e permitir a seleção delas.

### Importação

```js
import Radio from '@eduzz/houston-ui/Forms/Radio';
```

### Exemplo com Form

<Playground>
  {() => {
    const form = useForm({ defaultValues: { value: 1 } });
    return (
      <Form context={form} onSubmit={alert}>
        <p style={{ marginBottom: 8 }}>Valor: {JSON.stringify(form.watch('value'))}</p>
        <Radio.Group name='value'>
          <Radio.Item value={1} label='Opção 1' />
          <Radio.Item value={2} label='Opção 2' />
        </Radio.Group>
      </Form>
    );
  }}
</Playground>

### Exemplo sem Form

<Playground>
  {() => {
    const [value, setValue] = React.useState();
    return (
      <div>
        <p style={{ marginBottom: 8 }}>Valor: {JSON.stringify(value)}</p>
        <Radio.Group value={value} errorMessage='Mensagem de erro' onChange={setValue}>
          <Radio.Item helperText='Descrição do item' value={1} label='Opção 1' />
          <Radio.Item value={2} label='Opção' />
        </Radio.Group>
      </div>
    );
  }}
</Playground>

### Radio.Group props

| prop         | tipo                               | obrigatório | padrão | descrição               |
| ------------ | ---------------------------------- | ----------- | ------ | ----------------------- |
| name         | `string`                           | `false`     | -      | -                       |
| errorMessage | `string`                           | `false`     | -      | -                       |
| value        | `any`                              | `false`     | -      | Valor controle do grupo |
| onChange     | `function`                         | `false`     | -      | -                       |
| children     | `React.ReactElement<RadioProps>[]` | `true`      | -      | -                       |

### Radio.Item props

| prop       | tipo      | obrigatório | padrão | descrição               |
| ---------- | --------- | ----------- | ------ | ----------------------- |
| helperText | `string`  | `false`     | -      | -                       |
| value      | `any`     | `false`     | -      | -                       |
| disabled   | `boolean` | `false`     | -      | -                       |
| label      | `string`  | `false`     | -      | -                       |
| error      | `boolean` | `false`     | -      | Estado de erro do campo |
