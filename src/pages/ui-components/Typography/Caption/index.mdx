# Caption

O componente de caption é um elemento de legenda utilizado normalmente a uma figura ou ilustração.

### Importação

```js
import Caption from '@eduzz/houston-ui/Typography/Caption';
```

### Exemplo

<Playground>
  <figure style={{ display: 'inline-block', textAlign: 'center' }}>
    <img src='https://picsum.photos/250' alt='Random' />
    <Caption>Lorem ipsum</Caption>
  </figure>
</Playground>

### Informando ref e evento

<Playground>
  {() => {
    const ref = React.useRef(null);
    function scrollIntoView() {
      ref.current.scrollIntoView({
        behavior: 'smooth'
      });
    }
    return (
      <Caption ref={ref} onClick={scrollIntoView}>
        Click to scroll into view
      </Caption>
    );
  }}
</Playground>

| prop       | tipo                                                | obrigatório | padrão       |
| ---------- | --------------------------------------------------- | ----------- | ------------ |
| lineHeight | `Line Height Tokens`                                | `false`     | `md`         |
| className  | `string`                                            | `false`     | -            |
| onClick    | `(e: React.MouseEvent) => void`                     | `false`     | -            |
| as         | `'figcaption', 'p'`                                 | `false`     | `figcaption` |
| ref        | `React.ReactRef<HTMLParagraphElement, HTMLElement>` | `false`     | -            |
| color      | `'primary', 'low', 'pure'`                          | `false`     | `low`        |
