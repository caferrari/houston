# Typography

O componente de tipografia é utilizado para criação de títulos, subtítulos e textos, padronizando fontes, espaçamentos e tudo
referente a tipografias e hierarquia textual.

### Importação

```js
import Typography from '@eduzz/houston-ui/Typography';
```

### Informando ref e evento

<Playground>
  {() => {
    const ref = React.useRef(null);
    function scrollIntoView() {
      ref.current.scrollIntoView({
        behavior: 'smooth'
      });
    }
    return (
      <Typography ref={ref} onClick={scrollIntoView}>
        Click to scroll into view
      </Typography>
    );
  }}
</Playground>

| prop         | tipo                                                        | obrigatório | padrão    |
| ------------ | ----------------------------------------------------------- | ----------- | --------- |
| size         | `Font Size Tokens`                                          | `false`     | `xs`      |
| lineHeight   | `Line Height Tokens`                                        | `false`     | `md`      |
| weight       | `Font Weight Tokens`                                        | `false`     | `regular` |
| marginBottom | `boolean`                                                   | `false`     | -         |
| className    | `string`                                                    | `false`     | -         |
| onClick      | `(e: React.MouseEvent) => void`                             | `false`     | -         |
| as           | `'h1', 'h2', 'h3', 'h4', 'h5', 'h6', 'p', 'span', 'strong'` | `false`     | `p`       |
| ref          | `React.ReactRef<any>`                                       | `false`     | -         |
| color        | `'primary', 'low', 'pure'`                                  | `false`     | `low`     |
