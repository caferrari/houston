# Paragraph

O componente de parágrafo é utilizado para criarmos corpos de texto.

### Importação

```js
import Paragraph from '@eduzz/houston-ui/Typography/Paragraph';
```

### Informando ref e evento

<Playground>
  {() => {
    const ref = React.useRef(null);
    function scrollIntoView() {
      ref.current.scrollIntoView({
        behavior: 'smooth'
      });
    }
    return (
      <Paragraph ref={ref} onClick={scrollIntoView}>
        Click to scroll into view
      </Paragraph>
    );
  }}
</Playground>

| prop         | tipo                                                                   | obrigatório | padrão    |
| ------------ | ---------------------------------------------------------------------- | ----------- | --------- |
| size         | `sm, md, lg`                                                           | `false`     | `lg`      |
| lineHeight   | `Line Height Tokens`                                                   | `false`     | `md`      |
| weight       | `Font Weight Tokens`                                                   | `false`     | `regular` |
| marginBottom | `boolean`                                                              | `false`     | -         |
| className    | `string`                                                               | `false`     | -         |
| onClick      | `(e: React.MouseEvent) => void`                                        | `false`     | -         |
| as           | `'p', 'span', 'strong'`                                                | `false`     | `p`       |
| ref          | `React.ReactRef<HTMLParagraphElement I HTMLSpanElement I HTMLElement>` | `false`     | -         |
| color        | `'primary', 'low', 'pure'`                                             | `false`     | `low`     |
