# Heading

O componente de heading é utilizado para criação de títulos, fazendo uso da hierarquia textual e da semântica que o <i>HTML</i> fornece.

### Importação

```js
import Heading from '@eduzz/houston-ui/Typography/Heading';
```

### Tamanhos

Podemos definir o tamanho do Heading pela tag que passamos pela prop `as` ou forçando o tamanho pela prop `size`.

### Informando ref e evento

<Playground>
  {() => {
    const ref = React.useRef(null);
    function scrollIntoView() {
      ref.current.scrollIntoView({
        behavior: 'smooth'
      });
    }
    return (
      <Heading size='medium' ref={ref} onClick={scrollIntoView}>
        Click to scroll into view
      </Heading>
    );
  }}
</Playground>

| prop         | tipo                                                          | obrigatório | padrão |
| ------------ | ------------------------------------------------------------- | ----------- | ------ |
| size         | `'display', 'x-large', 'large', 'medium', 'small', 'x-small'` | `false`     | -      |
| lineHeight   | `Line Height Tokens`                                          | `false`     | `md`   |
| marginBottom | `boolean`                                                     | `false`     | -      |
| className    | `string`                                                      | `false`     | -      |
| onClick      | `(e: React.MouseEvent) => void`                               | `false`     | -      |
| as           | `'h1', 'h2', 'h3', 'h4', 'h5', 'h6'`                          | `false`     | `h1`   |
| ref          | `React.ReactRef<HTMLHeadingElement>`                          | `false`     | -      |
| color        | `'primary', 'low', 'pure'`                                    | `false`     | `low`  |
