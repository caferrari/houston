# Grid

O layout responsivo da grade do Houston se adapta ao tamanho e orientação da tela, garantindo a consistência entre layouts.

### Importação

```js
import Container from '@eduzz/houston-ui/Grid/Container';
import Column from '@eduzz/houston-ui/Grid/Column';
import Row from '@eduzz/houston-ui/Grid/Row';
```

### Conceitos

- **Container**: Utilizado para limitar o tamanho (width) do layout.
- **Row**: Utilizado para controlar as colunas.
- **Column**: Utilizado para limitar o tamanho do conteúdo interno.

### Alinhamento

Alinhamento vertical `alignItems`, alinhamento horizontal `justify`.

### Container Props

| prop    | tipo                | obrigatório | padrão  | descrição                                                       |
| ------- | ------------------- | ----------- | ------- | --------------------------------------------------------------- |
| spacing | `IContainerType`    | `false`     | `cozy`  | -                                                               |
| layout  | `IContainterLayout` | `false`     | `solid` | Se `fluid`, o valor da propriedade CSS `max-width` será `100%`. |

### Row Props

| prop       | tipo                 | obrigatório | padrão |
| ---------- | -------------------- | ----------- | ------ |
| alignItems | `GridItemsAlignment` | `false`     | -      |
| justify    | `GridJustification`  | `false`     | -      |

### Column Props

| prop | tipo       | obrigatório | padrão |
| ---- | ---------- | ----------- | ------ |
| xs   | `GridSize` | `false`     | `auto` |
| sm   | `GridSize` | `false`     | -      |
| md   | `GridSize` | `false`     | -      |
| lg   | `GridSize` | `false`     | -      |
| xl   | `GridSize` | `false`     | -      |
