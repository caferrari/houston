# Breadcrumb

O breadcrumb ajuda o usuário a visualizar a sua localização atual dentro da aplicação ou a sua localização em um contexto da aplicação, como etapas na criação de um produto. Também pode permitir o usuário clicar em uma etapa anterior para voltar para aquela localização.

### Importação

```js
import Breadcrumb from '@eduzz/houston-ui/Breadcrumb';
```

### Exemplo

<Playground>
  <Breadcrumb>
    <Breadcrumb.Link href='' icon={<HomeOutline />} />
    <Breadcrumb.Link href=''>Lista de Produtos</Breadcrumb.Link>
    <Breadcrumb.Link href=''>Produto</Breadcrumb.Link>
    <Breadcrumb.Item isActive>Edição de Produto</Breadcrumb.Item>
  </Breadcrumb>
</Playground>

### Separador customizado e sem links

<Playground>
  <Breadcrumb separator={<ArrowRight size='sm' />}>
    <Breadcrumb.Item>Home</Breadcrumb.Item>
    <Breadcrumb.Item>Lista de Produtos</Breadcrumb.Item>
    <Breadcrumb.Item>Edição de Produto</Breadcrumb.Item>
    <Breadcrumb.Item isActive>Livro: Mil Maneiras de ficar rico sem sair de casa</Breadcrumb.Item>
  </Breadcrumb>
</Playground>

### Exemplo com componentes externos (Link, NavLink, ...).

```jsx
import { NavLink } from 'react-router-dom';

<Breadcrumb>
  <Breadcrumb.Link as={NavLink} to='/' icon={<HomeIcon />} />
  <Breadcrumb.Link as={NavLink} to='/'>Lista de Produtos</Breadcrumb.Item>
  <Breadcrumb.Link as={NavLink} to='/'>Produtos</Breadcrumb.Link>
  <Breadcrumb.Item isActive> Livro: O Mistério</Breadcrumb.Item>
</Breadcrumb>
```

### Breadcrumb

| prop      | tipo              | obrigatório | padrão                           |
| --------- | ----------------- | ----------- | -------------------------------- |
| children  | `React.ReactNode` | `true`      | -                                |
| separator | `React.ReactNode` | `false`     | `<IconChevronRight size='md' />` |

### Breadcrumb.Item

| prop     | tipo              | obrigatório | padrão |
| -------- | ----------------- | ----------- | ------ |
| children | `React.ReactNode` | `false`     | -      |
| isActive | `boolean`         | `false`     | -      |

### Breadcrumb.Link

| prop     | tipo              | obrigatório | padrão |
| -------- | ----------------- | ----------- | ------ |
| children | `React.ReactNode` | `false`     | -      |
| icon     | `React.ReactNode` | `false`     | -      |
| as       | `React.ReactNode` | `false`     | `a`    |
| href     | `string`          | `false`     | -      |
