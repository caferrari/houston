# Breadcrumb

O breadcrumb ajuda o usuário a visualizar a sua localização atual dentro da aplicação ou a sua localização em um contexto da aplicação, como etapas na criação de um produto. Também pode permitir o usuário clicar em uma etapa anterior para voltar para aquela localização.

### Importação

```js
import Breadcrumb from '@eduzz/houston-ui/Breadcrumb';
```

### Exemplo

<Playground>
  <Breadcrumb>
    <Breadcrumb.Item>
      <Breadcrumb.Link href='' icon={<HomeOutline />} />
    </Breadcrumb.Item>
    <Breadcrumb.Item>
      <Breadcrumb.Link href=''>Lista de Produtos</Breadcrumb.Link>
    </Breadcrumb.Item>
    <Breadcrumb.Item>
      <Breadcrumb.Link href=''>Produto</Breadcrumb.Link>
    </Breadcrumb.Item>
    <Breadcrumb.Item isActive>Edição de Produto</Breadcrumb.Item>
  </Breadcrumb>
</Playground>

### Breadcrumb

| prop      | tipo              | obrigatório | padrão                           |
| --------- | ----------------- | ----------- | -------------------------------- |
| children  | `React.ReactNode` | `true`      | -                                |
| separator | `React.ReactNode` | `false`     | `<IconChevronRight size={14} />` |

### Breadcrumb.Item

| prop     | tipo                            | obrigatório | padrão |
| -------- | ------------------------------- | ----------- | ------ |
| children | `React.ReactNode`               | `false`     | -      |
| isActive | `boolean`                       | `false`     | -      |
| onClick  | `(e: React.MouseEvent) => void` | `false`     | -      |

### Breadcrumb.Link

| prop     | tipo              | obrigatório | padrão |
| -------- | ----------------- | ----------- | ------ |
| children | `React.ReactNode` | `false`     | -      |
| icon     | `React.ReactNode` | `false`     | -      |
| as       | `React.ReactNode` | `false`     | `a`    |
| href     | `string`          | `false`     | -      |
